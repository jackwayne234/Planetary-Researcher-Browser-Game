# Itch.io Release Notes

## Anomaly Discovery Update

Added a new anomaly investigation loop:

- Planets can now contain unusual signatures.
- Nearby unresolved signals can draw attention through the Anomalies tab.
- Use multiple instruments to gather evidence and confirm anomalies.
- Confirmed anomalies receive field IDs, map markers, and Fleet report options.
- Added a Glossary tab for instrument, HUD, anomaly, and map terminology.
- Added direct Save Reading support for instrument logs.
- Added planet name and sky phase to the HUD.

This update is meant to make exploration feel more like field research: notice something strange, cross-check it, confirm it, and decide what to record or report.
